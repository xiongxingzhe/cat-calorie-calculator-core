# Cat Calorie Calculator Core

Core TypeScript helpers for cat calorie planning based on veterinary RER/DER style formulas.

Built by [pawsandpounds.com](https://pawsandpounds.com).

## What this package includes

- `rerKg(kg)` - baseline resting energy requirement
- `dailyCatCalories(kg, profile)` - maintenance or weight-loss target kcal

## Install

```bash
npm i cat-calorie-calculator-core
```
